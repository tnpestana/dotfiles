# Ghostty Reference

## Key Bindings

### Window Management
- `Cmd+N` - New window
- `Cmd+T` - New tab
- `Cmd+W` - Close tab/window
- `Cmd+Q` - Quit
- `Cmd+1-9` - Switch to tab

### Splits
- `Cmd+D` - Split right
- `Cmd+Shift+D` - Split down
- `Cmd+[/]` - Navigate splits
- `Cmd+Opt+Arrow` - Resize split

### Text & Selection
- `Cmd+C/V` - Copy/paste
- `Cmd+F` - Find
- `Cmd++/-` - Increase/decrease font size
- `Cmd+0` - Reset font size

### Other
- `Cmd+K` - Clear scrollback
- `Cmd+,` - Open settings

## Configuration
- Theme: Catppuccin Macchiato
- Font: JetBrainsMono Nerd Font, 13pt
- Config: `~/.config/ghostty/config`

Note: Ghostty prioritizes `~/Library/Application Support/com.mitchellh.ghostty/config` over `~/.config/ghostty/config`
